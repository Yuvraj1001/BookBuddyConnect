## 📝 How to Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/Yuvraj1001/BookBuddyConnect.git
cd BookBuddyConnect

# 2. Create virtual environment
python -m venv venv
venv\Scripts\activate        # For Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the app
python main.py
